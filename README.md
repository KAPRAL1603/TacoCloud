# Taco Cloud

Spring Boot web application for designing and ordering tacos.

## Features
- Create custom tacos
- Choose ingredients
- Place taco orders

## Technologies
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf

## How to run
1. Open project in IntelliJ IDEA
2. Run TacoCloudApplication.java
3. Open http://localhost:8080
